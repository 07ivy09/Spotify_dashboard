# Project Overview :
This project is an end-to-end Power BI analytics solution built to analyze Spotify music streaming data across decades. It focuses on streaming performance, release trends, artist dominance, and audio features, presented through an interactive and business-ready dashboard. The dashboard is designed to simulate real-world analytics use cases relevant to Data Analyst/Business Analyst roles.

# Key Objectives :
1. Analyze total and average streams across tracks
2. Identify most streamed tracks and artists
3. Understand streaming trends by release year and month
4. Evaluate music attributes like energy, danceability, liveness, etc.
5. Enable interactive filtering by track, artist, and date
